## Contraintes techniques
- HTML statique (pas WordPress)
- Tailwind CSS v4 (CDN)
- Mobile-first
- One-page avec navigation par ancres
- Smooth scroll
- Animations subtiles au scroll
- Performance optimale
- SEO basique (meta, OG, schema)

## Direction artistique souhaitee
Palette inspiree du handpan: tons dores/cuivres, noirs profonds, blancs cremeux.
Typographie elegante et lisible.
Beaucoup d'espace blanc / breathing room.
Animations fluides, pas agressives.
Ambiance "son suspendu dans l'air".
